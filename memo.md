- インクリメント、デクリメントに ++/-- は存在しない。 +=/-= を使う

- / の結果は自動的に float になる。整数値を得たい場合は // を使う 

- 算術演算は「カッコ、累乗、（掛け算、割り算）、（足し算、引き算）」の順に優先される

- and, or は1つの文に複数回使用できる

- （用語）文には単純文と複合文がある。複合文の例は if 文。複合文は1行のヘッダー部分（条件判断部分とか）と、それに続くスイート部分で構成されている

- ヘッダー部分はコロンで終わる

- switch は無い。
