- コンテナには set という型もある。これは集合を扱うもので、要素の重複がなく、また、要素の順番を持たないという特徴がある。簡単な例は

```python
set1 = set([1, 2, 3])
set2 = set([1, 2, 3, 3, 1])
print(set1)
print(set2)

set1.add(4)
print(set1)

set1.remove(2)
print(set1)

set1.clear()
print(set1)
```
